1. Log Returns Matrix R

Convert daily closing prices into a T × n matrix of log returns. Log returns are time-additive and approximately normally distributed — essential for EWMA and the QP solver. T = 2089 rows (days), n = 39 columns (stocks).

Python
# Load prices, compute log returns
R = np.log(prices / prices.shift(1))
    .dropna()
# R.shape → (2089, 39) [T × n]
Rt,i = ln( Pt,i / Pt−1,i )
T = 2089 days · n = 39 stocks

2. EWMA — Exponential Weighting

Recent data matters more. EWMA assigns exponentially decaying weights with half-life h = 252 days. Data from 2018 carries <0.01% weight — old market regimes effectively vanish from the estimate.

Python
λ = np.exp(-np.log(2) / 252)
t = np.arange(T-1, -1, -1)
w = λ ** t
w /= w.sum() # normalise
λ = e−ln2 / h wk ∝ λk
h = 252 days · λ ≈ 0.9973
Decay weights (past → today)
3. Covariance Matrix Σ & Returns μ

Build the 39 × 39 symmetric PSD covariance matrix via EWMA-Cov(R) × 252. Σij captures how stocks i and j move together. The 39 × 1 vector μ gives annualised expected returns.

Python
ewm = R.ewm(halflife=252)
# Cov matrix (39×39, annualised)
Σ = ewm.cov().iloc[-39:].values * 252
# Expected returns (39×1)
μ = ewm.mean().iloc[-1].values * 252
Σ̂ = EWMA-Cov(R) × 252 μ̂i = EWMA-Mean(Ri) × 252
Σ is symmetric · Σ ≽ 0

4. Eigendecomposition — Spectral Theorem

Decompose Σ into eigenvalues Λ and eigenvectors Q. The Marchenko-Pastur threshold λ+ = σ̄²(1+√(N/T))² separates signal from noise. Only 4 of 39 eigenvalues are genuine market risk factors — the rest are statistical artifacts filtered out.

Python
eigenvalues, Q = np.linalg.eigh(Σ)
idx = np.argsort(eigenvalues)[::-1]
σ2 = np.mean(np.diag(Σ))
λ_plus = σ2 * (1+np.sqrt(N/T))**2
# → 4 signal / 35 noise factors
Σ = Q Λ Q⊤
λ+ = σ̄²(1+√(N/T))² · M-P bound

5. Constrained Optimisation — Quadratic Program

Minimise w⊤Σregw subject to target return, full investment, long-only, and 15% cap. Σreg = Σ + 0.05·I (Tikhonov). Returns stabilised with James-Stein shrinkage toward the mean. Solved via SLSQP.

Python
Σ_reg = Σ + 0.05*np.eye(n)
minimize(lambda w: w@Σ_reg@w,
  bounds=[(0,0.15)]*n, method='SLSQP',
  constraints=[{'fun': lambda w: w@μ-r_star},
              {'fun': lambda w: w.sum()-1}])
minw w⊤ Σreg w μ⊤w = r*  ·  1⊤w = 1 0 ≤ wi ≤ 0.15
